## üöÄ Get Started

Clone the repository:

```bash
git clone https://github.com/owenCTRL/Clickroot.git
cd Clickroot
```

Install dependencies:

```bash
npm install
# or
pip install -r requirements.txt
```

Start development server:

```bash
npm run dev
```
